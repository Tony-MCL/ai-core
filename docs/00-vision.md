# Vision

ai-core is a local-first agent platform for Morning Coffee Labs.

Petey is the first role: a strict, repo-aware code assistant that:
- reads the repo for holistic context (via indexing + targeted retrieval)
- proposes changes as patches (never silent overwrites)
- verifies with build/typecheck
- learns through transparent patterns/playbooks/snippets

ai-core is designed to support additional roles later (e.g., planning, analytics),
and to be embeddable into products like Progress in the future.
