# MVP Scope (v1)

## Must-have
- Repo indexing (file tree map, excludes)
- Search (ripgrep)
- Read files
- Propose plan
- Generate unified diff patch
- Apply patch (explicit confirm)
- Run verify command(s): tsc/build (configurable)
- Write task log + optional pattern suggestion

## Nice-to-have
- “Save as pattern/playbook” step after green build
- Repo map summary (entrypoints, packages, scripts)

## Out of scope (v1)
- model fine-tuning / LoRA training
- autonomous multi-step changes without confirmation
- multi-repo simultaneous operations
- network access by default
